MySchedule ICS Exporter Test Page

This page provides static test data for the MySchedule ICS Exporter browser extension. It mimics the structure of the MySchedule website to allow testing without requiring a login.

See the README.md for more info.

How to Use This Page

  1. Install the MySchedule ICS Exporter browser extension
  2. Navigate to this page
  3. Click on the extension icon in your browser toolbar
  4. The extension will load the mock data and allow you to export shifts as an ICS file
